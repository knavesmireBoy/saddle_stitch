Import PDF pages into InDesign for printing in booklet form. (Saddle Stich imposition format)

Typical usage is for printing of documents in portrait booklet form using an A4 (or - gosh you're posh - A3) printer instead of having a fistful of flat sheets to shuffle through. (Any landscape documents are not good candidates for imposition and best printed as is.) International Paper Sizes and millimetres are assumed. Now, about sucking eggs...

The InDesign document should be set to the twice the width of your PDF file. (File:Properties in Acrobat)

For a PDF document size of A5 create a new document to A4 landscape, for an A4 document, A3 landscape. Run the script. Saving the InDesign document is optional. Set SCALE TO FIT in the print dialog box along with correct orientation etc.. before printing.

Custom sizes are easily supported. For a portrait PDF document of 190 x 250 the DOCUMENT SIZE would need be 380 x 250. If you set a preset - or select existing - in the new Document dialog box on creating a new document there is no need to invoke new Document for subsequent PDF documents you intend to print.

If all goes well simply fold the ouputted sheets in half. Serving suggestion. A loose fitting rubberband may help with binding.

If you have a manual of say 200 pages you would be advised to split the document into no less than five sections of 40 pages, ten sheets per section. More typically this would be split per chapter (or two). Any pro version of Acrobat allows you to extract a range of pages to save as a separate document.

If using Preview on a Mac: First, open up the PDF file in Preview. Then select Print, and under Pages choose the range of pages you need to extract, ie (40-79). Then click Preview. Those pages will open up in a new window and you can select Save As from the file menu to save the extracted portion. But remember:

THE PDF TO IMPORT MUST HAVE PAGES IN MULTIPLES OF 4

To round up to a multiple of 4 INSERT BLANK pages with Acrobat or Preview. Other PDF editors are available.

Here are links to one, two and three page blank A5 Portrait pdf's which can be inserted at the end of your print-ready pdf. The size of these pages does not need to match your PDF document. The blanks are just there to allow the script to generate the correct page count in InDesign. Of course to save this hassle just include a few more pages from the next chapter to round up.

On running the script there will be FOUR PROMPTS. The first is for the OFFSET of the right-hand pages (defaults to 0 for left hand pages) this has a default of 210 but override at will. Leaving blank (deleting 210) will default to 148.5. The second prompt is for the PAGE COUNT, default is 16 leave blank for 40, override otherwise. It is trivial to alter these setting in the script, I've pointed out where.

After the prompt to place pages (accept the default) the CROP AREA needs to be set, this defaults to Media, but depending on how the pdf was produced, the setting may need to be one of:

  • Art
  • Bleed
  • Crop
  • Media
  • Trim

The script will run and generate the print ready InDesign document.

The script needs to saved as a JSX file in the following location (on a mac)

/Applications/Adobe InDesign */Scripts/Scripts Panel/Samples/JavaScript

In InDesign it will appear in Window:Utilities:Scripts (CS6)

Double-click to run the script, control/right click to edit in ExtendScript or...
